OWASP Juice Shop Vulnerability Assessment

This repository contains a cybersecurity internship project focused on performing vulnerability assessment and penetration testing on the OWASP Juice Shop application.

The assessment includes:

Automated security scanning
Manual verification
HTTP header analysis
Vulnerability identification
Risk assessment
Security recommendations
: Tools

OWASP ZAP
Nmap
Browser DevTools
: Topics Covered

SQL Injection
CORS Misconfiguration
Session Management Issues
Security Headers
Information Disclosure
Reconnaissance
